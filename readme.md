# couchdb-dat

Adapter between [dat](http://dat-data.com/) and [CouchDB](http://couchdb.apache.org/). Also works with [Cloudant](https://cloudant.com/)!

## Install

    npm install --save couchdb-dat

## Testing

    git clone https://github.com/garbados/couchdb-dat.git
    cd couchdb-dat
    npm install
    npm test
